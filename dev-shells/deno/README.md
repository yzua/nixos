# Deno

This is a minimal template for JavasScript development with Deno.

## Initialization

```bash
nix flake init -t "github:82163/nixos-config/main?dir=dev-shells#deno"
```

## Usage

- `nix develop`: opens up a `bash` shell with the required packages
