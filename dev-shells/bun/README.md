# Bun

This is a minimal template for JavasScript development with Bun.

## Initialization

```bash
nix flake init -t "github:82163/nixos-config/main?dir=dev-shells#bun"
```

## Usage

- `nix develop`: opens up a `bash` shell with the required packages
