# NodeJS

This is a minimal template for JavaScript/TypeScript development with Node.js.

## Initialization

```bash
nix flake init -t "/home/yz/System/dev-shells#nodejs"
```

## Usage

- `nix develop`: opens up a `bash` shell with the required packages

## Included tools

- `nodejs`, `pnpm`, `yarn`, `typescript`, `typescript-language-server`
- `prettier`, `eslint`
